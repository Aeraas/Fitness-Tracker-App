# Fitness Tracker (Java Console App)

Java project that tracks weight and workout entries using file storage.

## Current Features
- Add weight entries
- View weight entries
- Delete weight entries
- Add workout entries
- View workout entries
- Delete workout entries
- Save data to text files
- Load data on startup
- Input validation

## Concepts Used
- Classes and Objects
- Constructors
- ArrayList
- Methods
- File handling (FileWriter, Scanner)
- Basic input validation

## How to Run
1. Compile:
   javac FitnessTracker.java WeightEntry.java WorkoutEntry.java

2. Run:
   java FitnessTracker
